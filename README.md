# SpreadSheesh Talk

Talk about Tesserae.

## Run Slides

```bash
clojure -M -m sst.serve
```

## Dev

Start repl with :dev profile and run code in sst.serve 

Tailwind styles watcher

```
npx tailwindcss -i ./resources/styles.css -o ./resources/public/css/styles.css --watch
```
